a5e - Trojkat z x-ow

Status zadania: Pełna poprawność

Uwagi: Możliwe krótsze rozwiązanie



Przykładowe rozwiązanie:

a = input()
a = int(a)
przed = ""
for i in range(a):
	for j in range(0 , i + 1):
		przed = przed + "x"
	print(przed)
	przed = str("")


Rozwiązanie w języku Rust:

use std::io;

fn main() {

let mut a = String::new();

io::stdin().read_line(&mut a);

let a: i32 = a.trim().parse().expect("aaa");

let mut s = String::new();

for i in 0..a {
for j in 0..=i {
s = s.to_owned() + "x";
}
println!("{}", s);
s = String::new();
}
}
